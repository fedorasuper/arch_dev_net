# Arch DevNet Environment

This repository is designed for DevNet users who want a clean and minimal Linux environment for development and network automation.

Instead of using a preconfigured distribution, this setup is built on Arch Linux to maintain full control over the system. Arch provides a minimal base installation with zero bloatware, allowing users to install only the tools they need.

The goal of this repository is to create a lightweight, customizable, and efficient development environment tailored for DevNet engineers working with networking tools, automation frameworks, and scripting.

## Why Arch Linux?

* Minimal and lightweight base system
* No unnecessary packages or bloatware
* Fully customizable environment
* Ideal for building a personalized DevNet development workstation

## Target Users

* DevNet engineers
* Network automation engineers
* Linux users who prefer minimal systems
* Developers building network automation tools

This repository contains scripts, configurations, and documentation to help quickly set up an Arch-based DevNet development environment.
